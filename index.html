Jeu de la vie

  • #C
Voir la démo Voir le code

Application graphique du concept de "jeu de la vie" avec une gestion du vieillissement, calcul de la période d'oscillation, ainsi qu'une grille cyclique.

Le projet est intégralement codé en C, avec un make file pouvant généré la documentation intégrale grâce au générateur : "Doxygen".

Les différentes commandes applicables sont mises dans un README, le tout est disponible en open source sur github.

Démineur javascript

Voir le projet Voir le code

Le Démineur est un jeu de réflexion dont le but est de localiser des mines cachées dans une grille représentant, avec pour seule indication le nombre de mines dans les cases adjacentes.

Ce projet a été principalement codé en javascript avec un peu d'html et de css pour la disposition et le style assez simpliste.

Filtre de Sobel

  • Assembleur
Voir la démo Voir le code

Projet mips (assembleur) qui applique le "filtre de Sobel" sur des images bmp en 256x256.

Ce filtre consiste à détecter les contours d'une image grâce au gradient d'intensité des pixels.
